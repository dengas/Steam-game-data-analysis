# Steam-game-analytics

<div>
    <h2> Цель проекта: </h2>
    <p> Анализировать какие тэги в Steam проносят больше всего дохода и средней выручки </p>
</div>

<div>
    <h2> Что я сделал? </h2>
    <p> Анализировал датасет взятый из сайта Kaggle (url датасета находится в config) про Steam Games Dataset 2024. <br> Создал новый датафрейм: количество игр в тэге, общий доход (total revenue), средняя выручка (avg) </p>
</div>

<div><br>
  <h3> Инструменты, которые я использовал: </h3>
  <p><strong>Pandas</strong> — для работы с датасетом, и создания нового датафрейма</p>
  <p><strong>Plotly</strong> — для построения интерактивного графика</p>
</div>

<div><br>
    <h3> Трудности, с которыми я столкнулся: </h3>
    <p> Самое сложное в этом проекте было создать словарь подсчёта количество тэгов и их дохода </p>
</div>
